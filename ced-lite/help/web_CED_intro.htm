CED Lite Introduction

The Intel® EMGD Configuration Editor-Lite (CED-Lite) is a browser-based application that allows you to create configurations, that can be loaded directly on a Linux platform. Configurations are associated with a specific chipset and can be created for any one of the following supported chipsets:

• Intel® Atom™ Processor E6xx

• Intel® System Controller Hub US15W/US15WP/WPT chipset

Note: CED-Lite supports Intel® EMGD only on Linux. Windows, VBIOS, and EFI configurations are not supported in this application.

For complete information on using Intel® Embedded Media and Graphics Driver (Intel® EMGD), please see the Intel® Embedded Media and Graphics Driver User Guide.
