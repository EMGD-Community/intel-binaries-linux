Generating Intel® EMGD Configuration

After you have selected the desired configuration options, generate the configuration file by clicking Generate Intel EMGD Configuration.

1. Save the xorg.txt file to somewhere convenient such as your home directory.

2. In the /etc/X11 folder of the target Linux system, if there is an existing xorg.conf file, make a backup of it. You will need root permission to make changes in /etc/X11.

3. Move the xorg.txt file to /etc/X11 and rename it to xorg.conf.
