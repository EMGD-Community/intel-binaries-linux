Setting Overlay Color Correction

Overlay Color Correction allows the Overlay plane to have color-correction settings that are different from the main framebuffer color-correction settings (see Setting LVDS and SDVO Framebuffer Color Correction Attributes). This feature lets you color-correct for red, green, and blue, plus it enables you to adjust brightness, contrast, and saturation. The table below describes each of these settings.

Overlay Color Correction Values (applies to ALL color)

Option

Valid Range

Gamma:

0.6 to 6.0 (default value is 1)

Brightness:

0 to 200 (default value is 100)

Contrast:

0 to 200 (default value is 100)

Saturation:

0 to 200 (default value is 100)

 

Add your desired values to the correction fields or drag the controls to adjust values.
