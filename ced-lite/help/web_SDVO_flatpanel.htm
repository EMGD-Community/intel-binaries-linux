Configuring SDVO Flat Panel Settings

The Panel Settings tab in the SDVO Display (Advanced Settings) section allows you to specify settings for a flat panel display connected to an DDVO port. The table below describes each option on this tab.

SDVO Panel Settings Options

Flat Panel Settings

Description

SDVO Cards

Select the type of card(s) for your configuration to access attribute settings (see Port Driver Attributes for details). The choices are:

• Chrontel* CH7022 VGA/TV

• Chrontel* CH7307 DVI

• Chrontel* CH7308 LVDS

• Chrontel* CH7315 HDMI

• Chrontel* CH7317 VGA

• Chrontel* CH7319/CH7320 Dual-port DVI

• Silicon Image* SII1362 DVI

• Silicon Image* SII1364 DVI

• STM* IOH ConneXt

• OKI* ML7213

I2C Settings

The I2C Settings allow you to specify the I/O interface connections to devices on an SDVO port.

• I2C/DDC Bus Configuration: Speed (KHz) for the device. The range for these two fields is 10-400 KHz.

• Device Address Byte

— The I2C device address is for reading and writing device registers. The device address byte must be in 8-bit format with the 7-bit slave address assigned to its bits 7:1 and bit 0 set to 0.

— The DDC Device Address Byte is the I2C device address for reading EDID data from the display through the DDC bus.


*Other names and brands may be claimed as the property of others.
