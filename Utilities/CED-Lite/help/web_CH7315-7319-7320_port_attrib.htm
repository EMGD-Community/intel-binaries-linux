VGA BYPASS

9

Enables VGA bypass. To enable VGA Bypass, this configuration setting line must exist in the configuration file with the value of 2. Attribute 9 is used to enable selection of several possible display types based on what was supported on an SDVO device as defined in SDVO specifications. Default value of 2 represent VGA display.

1) Enable VGA Bypass
