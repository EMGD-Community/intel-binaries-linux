Port Driver Attributes

Standard Port Driver Attributes

Port drivers are modules within the Intel® Embedded Media and Graphics Driver that control SCH-specific modules such as SCH LVDS, SCH TV or add-on modules to SCH. The table below lists the attributes available to port drivers. Some of these standard attributes can be customized for specific port drivers and are detailed in the following topics:

In the attribute tables, device-specific (non-standard) attributes are highlighted in gray.

• Internal LVDS Port Driver Attributes (Mobile chipsets only)

• HDMI Port Driver Attributes

• Chrontel CH7307 Port Driver Attributes

• Chrontel CH7308 Port Driver Attributes

• Chrontel CH7315/CH7319/CH7320 Port Driver Attributes

• Chrontel CH7317b Port Driver Attributes

• Chrontel CH7022 Port Driver Attributes

• Silicon Image SiI 1362/SiI 1364 Port Driver DVI Attributes

• OKI* ML7213

• STM* IOH ConneXt

Note: Not all standard attributes are supported by all port drivers. Please see the topics mentioned above for details on the specific attributes supported by each port driver. Flat panel settings are specified via the FPINFO options of the configuration; please see For flat panel backlight timing settings, please see the Intel® Embedded Media and Graphics Driver User Guide.
